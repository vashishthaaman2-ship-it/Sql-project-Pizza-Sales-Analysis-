# Sql-project-Pizza-Sales-Analysis-
This project focuses on analyzing a pizza sales dataset using SQL to extract meaningful business insights. The objective of this project is to practice SQL querying skills and perform data analysis to understand sales patterns, customer preferences, and revenue performance.

---Key Analysis Performed

Some of the important SQL queries used in this project include:

.Total number of orders placed

.Total revenue generated

.Most ordered pizza types

.Highest revenue generating pizzas

.Sales distribution by pizza size

.Daily and hourly order trends

.Top 5 pizzas by quantity and revenue

.Category-wise sales performance
